System Analysis and Design

​For my System Analysis and Design project at Helena College, I collaborated with a team to optimize the judge assignment process for Montana Speech and Debate tournaments. The existing system, Tabroom, was complex and unreliable during large events, necessitating manual scheduling and judge assignments. Our analysis identified key areas for improvement, leading to the proposal of two third-party solutions: SpeechWire, offering tournament management tools with limited offline capabilities, and ArbiterSports Scheduling, a comprehensive scheduling system with mobile access and real-time updates. Additionally, we designed a mock-up system tailored to Montana's unique needs, featuring automated judge assignments, centralized event information, and real-time updates for organizers, coaches, judges, and students. This project demonstrated my ability to analyze existing systems, identify user needs, and design effective solutions to enhance operational efficiency.

Python Project

For my Python Project, I developed a versatile command-line personal assistant named Jarvis. Inspired by the character from Iron Man this application integrates various functionalities to assist users in daily tasks. Key features include:​

  • Voice Interaction: Utilizes the speech_recognition library to process voice commands, enabling hands-free operation.
  • Text-to-Speech: Employs pyttsx3 to provide auditory feedback and responses.
  • Email Management: Integrates with SMTP to send emails, supporting attachments and MIME formatting.
  • Web Interaction: Uses webbrowser for opening URLs and requests with BeautifulSoup for web scraping tasks.
  • Multimedia Playback: Incorporates pygame.mixer for audio playback and cv2 for video playback.
  • Hardware Integration: Supports Raspberry Pi GPIO for hardware control, including an LCD display and RFID reader.
  • Sensor Integration: Works with DHT11 sensors for environmental monitoring.

This project demonstrates my ability to integrate various Python libraries and hardware components to create a multifunctional assistant application and it also showcases my problem solving ability.
